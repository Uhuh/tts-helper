Multi Voices
This feature is designed to allow your users use multiple voices from any of the voices you add below. You must add the voices below, this allows you to restrict which voices users can use. Example usage from a user (imagine this is a Twitch redeem) /tts (brian): Hello world! I'm Brian! (ivy): And I'm Ivy! Any voice inside parenthesis will all be assumed lowercase, so casing in the name you set does not matter.
