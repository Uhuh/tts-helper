Chat Command Settings
Chat commands are good for Twitch streamers that haven't reached affiliate yet. Use the permission toggles to customize who can use the chat commands.
General Chat Settings
Enabled
If enabled, users can use this command to use normal TTS.
Command
The command that triggers the TTS.
Cooldown
How often this command can be ran in seconds.
Character Limit
How many characters that can be used in the chat message.
ChatGPT Chat Settings
Enabled
If enabled, users can use this command to generate ChatGPT TTS.
Command
Will prompt GPT to generate a response.
Cooldown
How often this command can be ran in seconds.
