Virtual Motion Capture Protocol
This is the VMC protocol, not the software. As it stands, TTS Helper's implementation will send the default blendshape parameters when sending mouth data.
  • "A" - To act as "mouth open".
  • "E" - To act as "mouth form".
Make sure the avatar's mouth has these parameters rigged with these blendshape names.
If you use VSeeFace, make sure to enable [VMC receiver] If testing doesn't work, unironically turn it off and on again.
